## Primero — Crear platos


mutation {
  createDish(input: {
    name: "Hamburguesa Clásica"
    description: "Carne de res, lechuga, tomate y queso"
    category: principal
    preparation_time: 15
  }) {
    id
    name
    description
    category
    preparation_time
    is_available
  }
}

## Cuarto — Consultar todos los platos con sus ingredientes y precios


query {
  dishes {
    id
    name
    description
    category
    is_active
    ingredients {
      id
      ingredient_id
      quantity
      unit
    }
    prices {
      id
      price
      valid_from
      valid_until
    }
  }
}




## Quinto — Consultar solo platos disponibles

## Séptimo — Actualizar un plato

mutation {
  updateDish(id: "1", input: {
    description: "Carne de res 200g, lechuga fresca, tomate y queso doble"
  }) {
    id
    name
    description
    updated_at
  }
}


## Octavo — Eliminar un ingrediente de un plato

mutation {
  removeDishIngredient(id: "1") 
}

Devuelve `true` si se eliminó correctamente, `false` si no encontró el registro.


## Noveno — Desactivar un plato


mutation {
  deactivateDish(id: "2") {
    id
    name
    is_available
  }
}


Luego vuelve a correr `dishes(onlyAvailable: true)` y verifica que ese plato ya no aparece.



Corre cada una en orden y cuéntame si algo falla.
